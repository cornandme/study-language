#### Note
* 변수에 선언한 타입과 다른 타입의 값을 넣으면 오류가 발생함.
* 두 변수의 타입이 다르면 연산할 수 없다.
* 한 번 선언된 변수와 같은 이름의 변수를 선언하여 갱신할 수 없다.
